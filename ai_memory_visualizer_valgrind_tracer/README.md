# AI Memory Visualizer + Valgrind Tracer — Programs Bundle

This bundle contains the C programs used in the laboratory project.

## Build

```bash
cd programs
make
```

## Run

```bash
./stack_example
./heap_example
./aliasing_example
./crash_example
```

## Notes

- Binaries are built with debug symbols (`-g`) for easier analysis.
- One program is expected to crash (`crash_example`).
